## 🔧 Troubleshooting Quick Fix

| Problem | Solution |
|---------|----------|
| Can't connect | Terminal open with mlflow ui running |
| No experiments | Run: `python -m pipelines.tuning_pipeline` |
| No nested runs | Scroll down on parent run page |
| Artifacts missing | Make sure `use_mlflow=True` was set |
| Port in use | Kill process: `taskkill /PID [id] /F` |

## 💡 Tips & Tricks

### Tip 1: Keep Terminal Open
MLflow server must keep running. Don't close the terminal!

### Tip 2: Use Chrome/Edge
Works best with modern browsers. Firefox also works fine.

### Tip 3: Refresh Browser
If something doesn't update, press F5 to refresh.

### Tip 4: Use Compare Feature
MLflow has a built-in comparison feature to compare runs side-by-side.

### Tip 5: Download JSON Results
The JSON files have all the detailed results you need.
